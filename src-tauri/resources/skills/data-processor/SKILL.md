# 数据处理工具

## 工作流程

1. 用户提供数据文件（CSV、Excel、JSON）
2. 分析数据结构和内容
3. 根据用户指令执行操作：
   - 数据清洗（去重、填充空值、格式标准化）
   - 格式转换（CSV ↔ Excel ↔ JSON）
   - 统计分析（汇总、分组、聚合）
   - 数据筛选和过滤
4. 输出处理结果

## 支持的操作

### 数据清洗
- 去除重复行
- 填充空值（默认值、均值、中位数）
- 格式标准化（日期、数字、文本）
- 异常值检测和处理

### 格式转换
- CSV → Excel
- Excel → CSV
- CSV/Excel → JSON
- JSON → CSV/Excel

### 统计分析
- 基本统计（均值、中位数、标准差）
- 分组汇总
- 交叉表
- 数据透视

## 输出

处理后的数据文件 + 处理报告（包含数据概览、处理步骤、结果统计）
